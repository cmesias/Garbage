[Summary]

A game where you throw yourself away because it is the right thing to do, or else a hoarder will keep you for eternity.

Submitted to Low Effort Game Jam at itch.io!

Link: https://cmesias.itch.io/


[Controls]
* WASD - Move
* H - Hide tips
* Escape - Exit Game
